Sobre mí

Tengo 20 años, soy de Buenos Aires, Argentina y me interesa la programación. Mi primero acercamiento al mundo de la programación fue en 2022, cuando mi madre me habló sobre unos cursos gratuitos que proporcionaba el Gobierno de la ciudad.

Empece con "Desarrolo de videojuegos en Unity" y, a pesar de haber terminado dicho curso, no me gustó tanto. Así que probé con un curso de "Desarrollo Web", donde conocí por primera vez HTML, CSS y JS. Acá fue donde me sentí mucho más comodo, ya que me parecía increible el poder crear una página web, hasta llegué a crear una versión muy básica de Mercado Libre, pero desafortunadamente perdí dicho proyecto. Tambien realicé un curso de Phyton pero no me enganchó tanto.

Yo creo que fueron todas esas experiencias las que me hicieron decantarme por la Tecnicatura Universitaria en Porgramación de la UTN, donde estoy a un año de recibirme. A lo largo de la carrera fui realizando varios proyectos, los cuales incluyen lenguajes como C++, C# y SQL Server, implementando programación estructurada, orientada a objetos y gestión de bases de datos respectivamente.

Proyectos

TPI-Prog2

Desarrollado en C++, este proyecto consiste en un sistema para gestionar turnos, pacientes y profesionales médicos. Incluye módulos para: registrar y almacenar datos de pacientes, médicos y turnos, manejar horarios y consultas (turnos), generar reportes, validar fechas y tiempos, uso de archivos para persistencia de datos. El objetivo fue integrar conceptos aprendidos en la materia, como estructuras de datos, modularización (múltiples archivos .cpp y .h), manejo de archivos de texto, y diseño de un menú interactivo para el usuario.

TP WinForm

El proyecto es una aplicación de escritorio desarrollada en C# con Windows Forms, orientada a la gestión de artículos de un catálogo comercial. Está pensada de manera genérica, de forma que pueda aplicarse a cualquier tipo de comercio, y permitir que la información almacenada pueda luego ser utilizada en diferentes servicios como páginas web, e-commerces, aplicaciones móviles o publicaciones impresas. Además, el sistema permite administrar de manera independiente las marcas y categorías disponibles, asegurando flexibilidad para adaptarse a distintos rubros comerciales. Toda la información se guarda de forma persistente en una base de datos existente, garantizando la integridad y disponibilidad de los datos.

IntegradorBD2

Este proyecto es un trabajo práctico integrador para la materia de Base de Datos II (BD2), centrado en la construcción y gestión de esquemas de base de datos con scripts SQL. El objetivo principal es diseñar, poblar, validar y extender una base de datos relacional mediante distintos objetos SQL como tablas, vistas, procedimientos almacenados y triggers.
